## Usage Notes:
### Global Search and Replace with Telescope and quickfix-reflector plugin
- <C-g> = Telescope live_grep
- <C-q> sends everything to quickfix window
- :s/search/replace/g
- :w 

- check nvim health
```
:checkhealth
```
